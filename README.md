## Projects

### [Ayojon — Multi-Vendor E-Commerce Platform](https://github.com/thetakibkhan)
Full-stack multi-vendor marketplace for event products in Bangladesh/South Asia.
- React 19 + TanStack Router/Query, Hono on Bun, oRPC, Drizzle ORM, PostgreSQL, Amazon S3
- Vendor onboarding, hybrid cart sync, multi-step bKash/COD checkout, dual-track order lifecycle
- Role-based access control, KPI dashboards, vendor/admin management

### [AI Ticket Processing Pipeline](https://github.com/thetakibkhan)
Async, AI-powered support ticket pipeline with two-phase processing and real-time status updates.
- Node.js + TypeScript + Express, PostgreSQL, AWS SQS (LocalStack), Portkey AI gateway, Socket.io
- Exponential backoff with jitter, DLQ routing on max retries, manual replay for failed tickets
- Zod-validated AI outputs, Vitest test coverage across worker, AI adapter, and Socket.io emitter

### [TaskOrbit — Micro-Task Earning Platform](https://github.com/thetakibkhan)
Full-stack micro-task marketplace connecting buyers and workers globally.
- React 19 + TanStack Router, Node.js + Express, MongoDB, Firebase Auth
- Role-based auth (worker/buyer/admin), coin-based payout flow, withdrawal management
